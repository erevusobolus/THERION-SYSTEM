# OpenClaw Overview: An Open-Source AI Automation Framework

**Last Updated**: 2026-02-03

## Introduction
OpenClaw is an open-source AI automation framework designed for developers who want full control over their AI workflows. Unlike hosted AI services, it runs on your infrastructure with complete source code access and customization through TypeScript plugins and configuration.

## Core Features
- Runs locally on user devices
- Integrates with messaging platforms (Telegram, WhatsApp, Slack, Discord, Signal, etc.)
- Supports extension channels like BlueBubbles, Matrix, and Zalo
- Enables AI agents to act on your behalf with zero external dependencies
